### About Project : 


- Make a simple and small with basic UI using the tailwind , react js and node js web app
- feature for it would be to take the URL from the User , 
- it do two things give a short URL for that url 
- also give two Qrcode one with the original url and one with short url 
- also a button to copy the short URL 
- QR code have download option in png and jpeg format 
- can change the size of the QR code or adjust the size of QR code if needed and download that version
- Project deployed on Fly.io
